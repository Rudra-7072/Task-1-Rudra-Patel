# Simple Chatbot

## Overview

This project is a basic text-based chatbot created using Python. It interacts with users through the command line and replies to a set of predefined messages. The conversation continues until the user enters `exit`.

## Features

* Command-line interaction
* Recognizes common greetings
* Provides responses for simple questions
* Returns a default message for unknown inputs
* Allows the user to end the chat with `exit`

## Built With

* Python 3

## Running the Project

### Step 1: Verify Python Installation

```bash
python --version
```

### Step 2: Start the Chatbot

```bash
python "Project 1.py"
```

If your computer uses `python3`, run:

```bash
python3 "Project 1.py"
```

## Sample Interaction

```text
🤖 Hey! I'm your friendly DecodeLabs chatbot. Let's talk!

You: hi
Bot: Hi there! What can I help you with today?

You: status
Bot: Everything is running smoothly on my end! How are you doing?

You: exit
Bot: Alright, shutting down. See ya!
```
